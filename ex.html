    User

    The User element contains the basis information about user

    properties
    Property Description
    id It's the identifier of a user
    email User email, this is also the username for login
    name User name
    lastaname User lastname
    created Indicates the date when user joined Repositorium
    active Indicates whether this user is active or not. Posibles values are 0 for inactive and 1 for active
