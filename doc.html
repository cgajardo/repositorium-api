    List of methods

    This table shows all available methods, * denotates parameter value, [...] denotates optional parameters, | stands for logic OR operator

    Method Resource Description
    GET /repositories Returns a list of all available repositories
    GET /repositories/*id Returns the respository with ID *id
    GET /repositories/*id/stats Returns stats from repository with ID *id
    GET /repositories/*id/search:*query[;fields:[*tags|*title|*content]] If currente users can afford for this download this method will retun a list of documents, otherwise it'll return a lis of challenges
