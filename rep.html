    List of Elements

    This page shows every element available and its representation

    Repository

    A repository es a main element in Repositorium, it represents a community of users and a set of documents related to that community

    properties
    id It's the identifier of a repository, you'll be using this in almost every request
    name Name of this repository.
    description A brief description of this repository
    author the author of this repository, this is an user element
    active Indicates whether this repository is active or not. Posibles values are 0 for inactive and 1 for active

    Authors and Contributors

    You can @mention a GitHub username to generate a link to their profile. The resulting <a> element will link to the contributor's GitHub Profile. For example: In 2007, Chris Wanstrath ( @defunkt), PJ Hyett (@pjhyett), and Tom Preston-Werner (@mojombo) founded GitHub.
